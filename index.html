Bienvenido al GitHub de Area51 Training Center.

Todos los instructores de Area51, relacionados con cursos que involucren código, utilizan GitHub para compartir parte de las clases con nuestros alumnos.

Si quieres descargar o ver parte del código realizado en nuestros cursos y estás acostumbrado a utilizar terminal, utiliza:

$ cd your_repo_root/repo_name
$ git clone repo_url
$ git status

Puedes buscar en Google clientes para GitHub en Windows o Mac, aunque siempre recomendaremos usar el terminal ;).

Si no estás acostumbrado a usar GitHub y quieres decargar el código, ingresa a uno de nuestros repositorios, en la columna de la derecha encontrarás: Download Zip.

DISCLAIMER

El código de nuestro GitHub no siempre está 100% optimizado para producción, simplemente por temas pedagógicos. Úsalo con prudencia y sólo para fines educativos.

¿Qué cursos podré encontrar?

Publicaremos y actualizaremos el código de nuestros cursos todos los días, eso quiere decir que puedes encontrar cursos completos de HTML5, iOS, Android, Python, Ruby, etc. Eres libre de compartir o publicar nuestros repositorios :).
